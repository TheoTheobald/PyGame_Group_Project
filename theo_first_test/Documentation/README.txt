README

Set up:
1. Open file Explorer and unzip file.
2. Open Python IDE of choice.
3. Click 'File' and 'Open' and select all files in unzipped folder to open.
4. Run'main.py' to play.
5. Change game settings in 'settings.py'.

Requirements:
1. Python IDE 3.8.8

Libaries:
1. pygame 2.0.0
2. sys 3.8
3. os 3.7
4. random 3.10

OS:
1. WIN 10
2. MacOS

Recommended screen resolution:
1280 x 768 pixels
